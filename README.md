# WhatsFlow Portfolio

WhatsFlow is a WhatsApp CRM and automation platform portfolio project built with Laravel.

## Features

- Dashboard Analytics
- Lead Management
- Conversation Inbox
- Campaign Management
- WhatsApp Templates
- Auto Replies
- Knowledge Base
- Portfolio Demo Mode

## Tech Stack

- Laravel 12
- PHP 8.2+
- MySQL
- Tailwind CSS
- Chart.js
